# 🏪 Retail Decision Engine

> Demand Forecasting × Dynamic Pricing × Inventory Optimization — all in one dashboard.

A Streamlit app that combines Deep Learning and Mathematical Optimization models to give grocery and retail managers a weekly decision brief: what price to set, how much demand to expect, and when to reorder.

## Features

### 📈 Demand Forecasting (Deep Learning)
- Powered by **Temporal Fusion Transformer (TFT)** (`pytorch-forecasting`), trained on 3 years of daily synthetic sales data.
- Captures complex non-linear relationships between pricing, seasonality, and macro trends.
- Real-time dynamic inference using cached PyTorch models (`.ckpt`) for seamless UX.

### 💰 Dynamic Pricing
- AI-driven demand simulation: Interactive price slider feeds directly into the TFT model to predict future demand dynamically.
- SciPy optimizer calculates the theoretical profit-maximizing price benchmark.
- Hybrid Rendering Visualization: Compares the theoretical price elasticity curve against the AI's 1-point predicted demand and profit.

### 📦 Inventory Optimization
- OR-Tools CP-SAT solver plans optimal order quantities for the next 8 weeks.
- **Fully Integrated Pipeline:** The inventory solver directly uses the dynamic demand forecast generated by the TFT model based on the user's selected price.
- Hard constraints: maximum warehouse capacity, lead time.
- Soft constraints & Objective: Uses slack variables to model safety stock violations and stockouts as penalty costs. The solver minimizes total cost (order fixed cost, holding cost, plus penalties) to ensure mathematical feasibility even in extreme edge cases.

### 🎯 Summary Dashboard
- Integrates all three modules into one actionable view per product.
- Real-time state management: Updates KPIs (Demand, Revenue, Profit impact) instantly based on the price scenarios tested in the individual product tabs.
- Gap analysis: Recommends actionable pricing strategies (Raise/Lower/Keep) by comparing current settings with optimal targets.

## Getting Started

### Prerequisites
- Python 3.10+ (Tested on 3.11/3.12)
- pip / Docker

### Installation

```bash
git clone [https://github.com/mona2083/retail-decision-engine.git](https://github.com/mona2083/retail-decision-engine.git)
cd retail-decision-engine
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## Usage

### Demo Data
1. Click **"🎲 Generate & Run Models"** in the sidebar to generate 3 years of synthetic daily data and run all models.
2. Change the **Random Seed** to generate a different dataset.
3. Select a product tab (🥛 Milk / 🍞 Bread / 🍊 Orange Juice).

### Scenario Simulation (End-to-End Pipeline)
1. In the **Dynamic Pricing** section, adjust the price slider. Watch the AI predict the new demand curve.
2. Go to the **📦 Inventory Optimization** section within the same tab.
3. Click **"🚀 Optimize Order Plan"**. The OR-Tools solver will calculate the order plan based on the *AI's newly predicted demand* from your price setting.
4. Results and recommended actions appear in the **🎯 Summary Dashboard** at the bottom of the page.
